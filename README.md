# Chat-Room-Sederhana-Menggunakan-Protocol-TCP
Tugas membuat chat room sederhana dengan Protocol TCP (transmission Control Protocol) Mata Kuliah Jaringan Komputer Kelas A

Andi Abdul Malik Imaduddien - 5024241087

## Cara Menjalankan

### 1. Jalankan Server

Buka terminal pertama:

```bash
python server.py
```

Output:

```text
Server berjalan di port 5000
```

---

### 2. Jalankan Client

Buka terminal baru:

```bash
python client.py
```

Masukkan nama dan IP server:

```text
Masukkan nama: Andi
Masukkan IP Server: 127.0.0.1
```

---

### 3. Menjalankan Banyak Client

Buka beberapa terminal dan jalankan:

```bash
python client.py
```

---

## Penggunaan Dalam Satu Laptop

Gunakan IP:

```text
127.0.0.1
```

atau

```text
localhost
```

Karena server dan client berjalan pada perangkat yang sama.
